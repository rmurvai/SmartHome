# SmartHome
Application to read some data from sensors 

## What you will need
* Arduino Uno
* DHT22 temperature/humidity sensor
* Pir sensor for motion detection
* MQ-2 sensor for LPG, CO and smoke detection
* one led
* breadboard 
* wires
